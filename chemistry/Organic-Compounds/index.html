    # Groups

    1. the prefix indicates the quantity of carbons in a component
    2. the suffix indicates the presence of substituent groups

    # Branches

    1. name the longest carbon chain
    2. number the carbons so the first substituent group has the lowest possible index
    3. combine duplicate groups with format a,b,c-<prefix><group>, using numerical prefixes (di, tri, tetra, etc.)
    4. order these groups by alphabetical order, ignore numerical prefixes

    # Example

    1. the longest carbon chain is 9 long nonane
    2. number from right to left, so the methyl group has the lowest index
      • 2,4-dimethyl
      • 6-isopropyl more details on alkyls here
    3. order alphabetically, 6-isopropyl 2,4-dimethyl nonane

    # Functional Groups

    • alkynes :: contain at least $1\space\text{C}\equiv\text{C}$ (ex: 1-butyne)
    • alkenes :: contain at least $1\space\text{C}=\text{C}$ (ex: 2-pentene)
      • branched alkenes
        • 3-ethyl hexane
        • 2,3,4,7-tetramethyl octane
    • alcohol :: exists as $\text{R}-\text{OH}$ (ex: ethanol, 1-butanol)
    • ether :: exists as $\text{R}-\text{O}-\text{R}’$ (ex: diethyl ether, ethyl methyl ether)
    • ketone :: contains $\text{C}=\text{O}$ bond in middle (ex: 2-pentanone)
    • aldehyde :: contains $\text{C}=\text{O}$ bond at end (ex: pentanal)
    • amines :: exists as $\text{R}-\text{NH}_2$ (ex: methylamine, trimethylamine)
    • carboxylic acids :: exists as $\text{R}-\text{COOH}$
      • ethanoic acid :: acetic acid
      • methanoic acid :: formic acid
      • butanoic acid :: butyric acid
    • ester :: exists as $\text{R}-\text{COO}-\text{R}$; smells nice (ex: methyl butyrate, ethyl acetate)
